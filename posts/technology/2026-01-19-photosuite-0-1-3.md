```typescript
import { photosuite } from 'photosuite';

photosuite({
  scope: 'main',
  // ... 其他 photosuite 配置

  // 自定义 Fancybox 配置
  fancyboxOptions: {
    Carousel: {
      infinite: true,
      Thumbs: false,
      Toolbar: {
          display: {
            left: ["counter"],
            right: ["autoplay", "close"]
          },
      },
    }
  }
})
```

这样一来，不管图片比例多么悬殊，它们在拼图中都会自然对齐

**高度一致、宽度合理、边缘整齐**，同时也避免了任何形式的拉伸或裁剪

### 布局样式

样式只负责布局：Flex + gap + 响应式间距，没有额外装饰，所有空间都留给图片本身

这里通过 CSS 变量统一管理间距；在移动端则适当缩小 gap，以保证有限屏幕宽度下的视觉紧凑度

```scss
// src/style/fancybox.scss

/**
 * @file demo.ts
 * @description 演示页面的入口脚本，用于展示 Photosuite 的功能
 */

import './style.css'
import { photosuite } from './index'

// 初始化 Photosuite，启用所有功能并配置 Fancybox 选项
photosuite({
  scope: 'main',
  fancyboxOptions: {
    Carousel: {
      infinite: false,
      Thumbs: false,
      Toolbar: {
        display: {
          right: ["close"]
        },
      },
    }
  }
})
```

## 安装

Photosuite 已发布至 npm，可直接安装：

```bash
pnpm add photosuite
# or
npm install photosuite
# or
yarn add photosuite
```
